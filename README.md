# report-beacon-flood

● Beacon Flood 공격을 날릴 수 있는 프로그램임.


## 기능
● 랜카드 Monitor Mode (모니터 모드) 자동 전환.

● SSID를 사전(Dictionary) 파일화하여 공격에 사용함.

● 한국어 SSID 지원.

● WIFI를 사용하는 모든 기종에 공격을 지원함.
